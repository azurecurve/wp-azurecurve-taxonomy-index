== Description ==
Displays Index of Categories/Tags or other taxonomy types using taxonomy-index Shortcode. This plugin is multi-site compatible.

Example use: [taxonomy-index taxonomy='category' slug='ice-cream']

Alternative ti shortcode can also be used

Taxonomy can be set to category or tag or other taxonomy post type.

== Installation ==
To install the plugin copy the <em>azurcurve-taxonomy-index</em> folder into your plug-in directory and activate it.
